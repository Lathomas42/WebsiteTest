PyQtGraph Examples

PyQtGraph

PyQtGraph is a python graphics and GUI library intended for mathematics/engineering applications. This Library is very fast due to implimenting numpy for the number crunching.

Pros: Cons:
-Very Fast -Overly complex
-Written entirely in python
Best Uses
PyQtGraph is best used for visualizing large data sets quickly. It is the fastest package although the lack of a full documentation and complication makes this package undesirable for simple plots.
Installation
  • Windows: download and run the .exe installer
  • Mac/Linux: Download the .tar.gz file and run "python setup.py" from within the directory of the extracted files.
Bokeh Examples

Bokeh

Bokeh is a python visualization library mainly targeted at modern browsers for presentations. Bokeh quickly creates interactive plots,

Pros: Cons:
-Widely used and accepted -Not the fastest
-Well Documented
Best Uses
Bokeh is an easy to use library and should be used if having an interactive html result is desired. This means that your plot can be interactive on machines with no Python installed, just send the html file. This feature alone makes Bokeh stand out.
Installation
  • Dependencies:
    • NumPy
    • Flask
    • Redis
    • Requests
    • gevent
    • Pandas
  • From Anaconda:(Recommended) simply run the command: "$ conda install bokeh"
  • From PyPi: Run the command "$ pip install bokeh"
Matplotlib

Matplotlib

Matplotlib is the most widely used python 2D plotting library. Matplotlib attempts to mimic the style of MATLAB plots and MATLAB's interface.

Pros: Cons:
-Widely used and accepted -Not the fastest
-Well Documented
Best Uses
Matplotlib is the go-to library for python visualization and for good reason. Matplotlib provides quick, easy, and expected plots with great documentation and a helpful community to back it up. If you are not sure where to start, start with Matplotlib.
Installation
  • Linux: Install through your package manager. Run the command "sudo apt-get install python-matplotlib" (Debian/Ubuntu) or "sudo yum install python-matplotlib" (Fedora/Redhat)
  • Mac: Open terminal and run the command "sudo port install py27-pip" followed by "sudo pip-2.7 install matplotlib"(note if pip already installed the first command can be omitted)
  • Windows:Either run python through Enthought Canopy or Continuum Anaconda (both free) or install from Matplotlib's Github source code

This example shows off the ease of Bokeh to create very nice looking charts that are easily interactive.

In Bokeh, when you create a plot, instead of bringing up a python windows to display the png of the chart, Bokeh saves the plot to a html file that allows you to interact with the plot through your browser.

This specific example is the closing stock prices for a few major companies plotted against each other.

The data for this specific plot was found through Bokeh's easy to access sample data sets. These data sets can be accessed by importing bokeh.sampledata. Using bokeh's sample data is a great way to introduce yourself to data visualization because they have examples for each dataset, however, you can take the data and visualize it in a different way.

import numpy as np
from bokeh.sampledata.stocks import AAPL, FB, GOOG, IBM, MSFT
from bokeh.plotting import *

def bokehLinePlot():
	output_file("stocks.html", title="stocks.py example")

	hold()

	figure(x_axis_type = "datetime", 
	       tools="pan,wheel_zoom,box_zoom,reset,previewsave")
	# easy way to make line chart in bokeh
	line(np.array(AAPL['date'], 'M64'), AAPL['adj_close'],
	     color='#A6CEE3', legend='AAPL')
	line(np.array(FB['date'], 'M64'), FB['adj_close'],
	     color='#1F78B4', legend='FB')
	line(np.array(GOOG['date'], 'M64'), GOOG['adj_close'],
	     color='#B2DF8A', legend='GOOG')
	line(np.array(IBM['date'], 'M64'), IBM['adj_close'],
	     color='#33A02C', legend='IBM')
	line(np.array(MSFT['date'], 'M64'), MSFT['adj_close'],
	     color='#FB9A99', legend='MSFT')

	curplot().title = "Stock Closing Prices"
	grid().grid_line_alpha=0.3

	aapl = np.array(AAPL['adj_close'])
	aapl_dates = np.array(AAPL['date'], dtype=np.datetime64)
	return curplot()
	window_size = 30
	window = np.ones(window_size)/float(window_size)
	aapl_avg = np.convolve(aapl, window, 'same')
	show()  # open a browser

Some popular uses of Networkx

  • Simple Directed Graph
  • Data

In this example, a simple directed graph is constructed from data made up of strings saying who is friends with who. This example involves nodes, which are people, and links, showing who is friends with who.

Directed Graphs can be used in many different applications, whenever the links have the possibility of being one-directional. Friendship is a bit of a cruel example of this (notice Bill is friends with Sue but Sue is not friends with Bill). Other examples of applications of directed graphs are networks like the world wide web, where a website has a directed edge to another website if there is a link from the website to the other website.

The data I used for this was completely made up data of a friends group that was saved in a ".txt" file and looked like this:

Bill is friends with Joe,Sue,Mary
Sue is friends with Joe,Mary
Joe is friends with Sue,Bill
Mary is friends with Sue,Bill

This was not the most organized way to keep this code but was used as an example to show that you can parse edges from anything. 

import networkx as nx
# showing a graph in networkx is done through matplotlib
import matplotlib.pyplot as plt 


def GraphFriends():
	friendships = [] # initialize empty friendships list
	f = open('friends.txt','r') # open data file
	for line in f.readlines():
		# each line in f.readlines() is of the form "A is friends with B,C,D"
		line = line.split(' is friends with ')# splits to [A,[B,C,D]]
		id = line[0] # A
		friends = line[1] # [B,C,D]
		for friend in friends.split(','): 
			friend = friend.rstrip()
			friendships.append([id,friend]) # Appends [A,B],[A,C],[A,D]
	f.close() # close the file that was opened
	
	Friends_Graph = nx.DiGraph() # initialize empty Directe Graph
	Friends_Graph.add_edges_from(friendships) # Adds edges from the list that was made
	nx.draw(Friends_Graph) # draw in Networkx
	plt.draw() # draw in matplotlib
	plt.show() # show the plot

Matplotlib Examples

Some popular uses of Matplotlib

  • Stacked Bar Plot

In this example I used Matplotlib's bar plot function to show a large dataset, all Olympic athletes. Specifically showing how many medals were won in each age group.

In the bar graph, the different colors represent the breakdown of golds,silvers, and bronzes inside the total amount of medals. This was done by using the fact that in matplotlib, the first thing you call to be plotted will be under everything else. So I plotted all medals won as a gold bar plot, then all bronze and silver medals as a silver plot, then all bronze medals as a bronze plot.

The library Numpy was also used in this example. Numpy is a widely used python library that allows very fast calculations using numpy-arrays. Whenever you are working with a large data set, use numpy to speed things up.

The data used for this example is a free data-set from Sports Reference who has a large (8618 athlete) dataset for the history of the olympics.

This data was already formatted in a .xlsx format though I switched it to a .csv type to be easily readable in python. The csv was formatted in columns like so:

Athlete,Age,Country,Year,Closing Ceremony Date,Sport,Gold Medals,Silver Medals,Bronze Medals,Total Medals

Although this was way too much information for what I wanted to do so I only payed attention to the Age and the Medals Columns, ignoring 5 columns.

There are thousands of free data-sets on-line, this is a great example of one of them but many more can be found in places like data.gov. A word of warning with free data-sets, they often have too much information, such as this case, where we ignored many of the columns, The unnecessary information should be thrown away at the start to reduce memory used.

import numpy as np
from matplotlib import pyplot as plt

def PlotOlympicHist():
	# first open the file with the data you wish to plot
	f = open('OlympicAthletes.csv','r')
	# numpy includes a useful genfromtxt function which allows you to
	# call your data by column
	data = np.genfromtxt(f, delimiter=',',skip_header=1, 
                         names=['Athlete','Age','Country','Year',
                                'Closing Ceremony Date','Sport',
                                'Gold','Silver','Bronze','Total'])
	# initializing the bins for the histogram, a span of the ages
	bins = np.arange(min(data['Age']),max(data['Age'])+1)
	# close files after you read the data
	f.close()
	# initialize arrays for Golds/Silvers/Bronzes
	Golds=[]
	Silvers=[]
	Bronzes=[]
	for age in bins:
		# fill each array with the total amount of medals won by people
		# of that age
		age_golds = data['Gold'][data['Age']==int(age)]
		age_silvers = data['Silver'][data['Age']==int(age)]
		age_bronzes = data['Bronze'][data['Age']==int(age)]
		Golds.append(sum(age_golds))
		Silvers.append(sum(age_silvers))
		Bronzes.append(sum(age_bronzes))
		
	# matplotlib allows you to change xlabel, ylabel, and title with these commands
	plt.xlabel('Age')
	plt.ylabel('Total Medals Won')
	plt.title('Total Medals Won Vs Age of Athlete')
	
	# using a bar plot to show how many medals were won by each age group
	plt.bar(bins,np.add(np.add(Golds,Bronzes),Silvers),color="#E6B800")
	plt.bar(bins,np.add(Silvers,Bronzes),color="#999999")
	plt.bar(bins,Bronzes,color="#993D00")
	plt.grid(True)
	plt.show()

In this example I used PyQtGraph's ScatterPlotItem() Class which behaves as a normal Scatter Plot should. The Plus side for PyQtGraph is that it is fast and deals very well with large Datasets. For this reason I used two large datasets (described in the data tab). Making 4 scatter-plots with the datasets, each scatter plot comparing 2 variables (x and y axis) and a third using the color of the nodes.

The left column of scatter plots are Red Wine comparisons and the Right Column are white wine comparisons. The top plots are pH vs Alcohol content and the bottom plots are Residual sugar vs Density. All of the nodes are colored according to their quality rating, the scale can be seen in the bottom right graph.

The data used in this plot are large data-sets downloaded from UCI's machine learning database, they are two datasets, preformatted in csv form. One set is red wine(~1600 entries), one set is white wine(~4900 entries). Both sets are similarly formated into columns according to :

"fixed acidity";"volatile acidity";"citric acid";"residual sugar";"chlorides";"free sulfur dioxide";"total sulfur dioxide";"density";"pH";"sulphates";"alcohol";"quality"

UCI is another place on the internet to get great sets of data for free. This particular set allowed me to compare different qualities of red and white wine to the rating to look at trends. Note that in this example, there were many different comparisons I could have made between qualities. The top comparison seems to be a decent comparison because we see a clear trend in rating as we go to higher pH/alchohol content. The bottom plots seem to be poor comparisons because there is no real trend going on.

Sometimes when you have large sets of data with many different qualities it requires you to hunt around and try some different comparisons or plot types before you get one that has a significant trend.
